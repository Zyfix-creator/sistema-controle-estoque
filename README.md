# 📦 Sistema de Controle de Estoque

Projeto desenvolvido em Java utilizando Programação Orientada a Objetos para gerenciamento de produtos em estoque.

## 🚀 Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Busca por código
- Atualização de quantidade
- Remoção de produtos

## 🛠️ Tecnologias

- Java
- Programação Orientada a Objetos (POO)
- Git
- GitHub

## 📂 Estrutura

```text
src/
├── Main.java
├── Produto.java
└── Estoque.java
```

## ▶️ Como executar

1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute o arquivo Main.java
